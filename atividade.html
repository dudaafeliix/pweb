Questão 1: Descrição do CSS

a) Classe .caixa

background-color: lightblue, é para definir a cor de fundo como azul claro; width: é para definir a largura em 300px; height: é para definir a altura em 150px; border: serve para adicionar uma borda sólida na cor azul claro com 2px; padding: é para adicinar um espaço interno em 20px; margin: é para adicinar um espaço externo em 30px; color: define a cor do texto como azul escuro (darkblue); text-align: center; serve para centralizar o texto dentro da caixa.

Esta é uma caixa estilizada!

b) Classe .aviso

background-color: define a cor de fundo como um tom claro de vermelho; border: cria uma borda tracejada vermelha de 1px; padding: 10px; adiciona um padding de 10px margin: 50px auto; define uma margem de 50px acima e abaixo, e centraliza horizontalmente com auto width: define a largura como 400px; text-align: left; alinha o texto à esquerda; color: #990000; define a cor do texto como vermelho escuro.

Atenção! Este é um aviso importante.
